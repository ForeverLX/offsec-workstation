# NightForge Shell Migration: DMS → Quickshell + Matugen

## Overview
Migration from Dank Material Shell (DMS) to modular Quickshell + Matugen stack on Niri WM.

**Target Aesthetic**: [ilyamiro/nixos-configuration](https://github.com/ilyamiro/nixos-configuration)
- Matugen-driven dark/light themes
- JetBrains Mono + Iosevka Nerd fonts
- Rofi launcher, SwayOSD overlays
- Cava visualization (optional)

## Why Migrate?
- **DMS Dependency**: Heavy Electron-based shell, resource-intensive
- **Modularity**: Quickshell QML = native Wayland, compositor-integrated
- **GPU Constraints**: GTX 1650 benefits from lighter stack
- **Operator Integration**: Direct Niri IPC for VPN/engagement status

## Pre-Migration Checklist
```bash
# 1. Backup existing configs
cd ~/.config
tar -czf nightforge-backup-$(date +%Y%m%d).tgz niri ghostty rofi swayosd

# 2. Verify Nvidia drivers
nvidia-smi  # Should show GTX 1650

# 3. Check Niri version (need 0.1.9+)
niri --version
```

## Installation
```bash
# Install packages (from updated manifests/host-packages.txt)
yay -S quickshell-git matugen-bin rofi-wayland swayosd awww \
         ttf-jetbrains-mono-nerd ttf-iosevka-nerd cava cliphist

# Or use pacman for repo packages:
sudo pacman -S rofi-wayland swayosd ttf-jetbrains-mono-nerd
```

## Key Changes from DMS

### Removed
- `spawn-at-startup "dms" "run" "-d"` (DMS autostart)
- All `dms ipc call` keybinds (spotlight, clipboard, processlist, etc.)
- DMS config includes (`include "dms/*.kdl"`)

### Added
- **Quickshell**: Native Wayland shell with QML
  - Workspace indicators (Niri IPC)
  - Clock + battery + VPN status
  - Podman container status
  - Engagement context display

- **Rofi-wayland**: Application launcher (replaces DMS spotlight)
  - `Mod+D` to launch
  - Clipboard mode via cliphist (`Mod+V`)

- **SwayOSD**: On-screen overlays (replaces DMS audio/brightness)
  - Volume/brightness indicators
  - Media player info

- **Matugen Pipeline**:
  ```bash
  Wallpaper → matugen generate → /tmp/matugen/colors.json
            ↓
  matugen-sync.sh → Update: Ghostty, Rofi, Quickshell, env vars
  ```

## Deploy
```bash
cd ~/Github/nightforge
bash scripts/deploy.sh
```

This stows all dotfile packages and ensures scripts are executable.

## Systemd Services

### MPD
```bash

systemctl --user start mpd.service
systemctl --user status mpd.service
```

### Matugen Sync
```bash
systemctl --user enable matugen-sync.service
```
Triggers once on graphical session start.

### Wallpaper Rotation
```bash
systemctl --user enable --now wallpaper-rotate.timer
systemctl --user status wallpaper-rotate.timer
```

## Wallpaper Rotation (DMS Replacement)
DMS had 15-minute auto-switch. Replaced with:
1. **Manual trigger**: `Mod+Alt+R` (Niri keybind)
2. **Systemd timer**: `wallpaper-rotate.timer` (every 15 min)
3. **Script**: `scripts/wallpaper-rotate.sh`

Enable timer:
```bash
systemctl --user enable --now wallpaper-rotate.timer
```

## Operator Features (Unique to NightForge)
- **VPN Status**: WireGuard mesh status in Quickshell bar
- **Engagement Context**: Current engagement display (`Mod+Shift+G`)
- **Podman Status**: Container count in bar, details (`Mod+Shift+T`)
- **MITRE Logging**: Integrated with operator-terminal modules

## Troubleshooting

### Niri won't start
```bash
niri validate                           # Check config syntax
journalctl --user -u niri.service -n 50
```

### Quickshell not showing
```bash
quickshell --config ~/.config/quickshell/main.qml
# Check logs: ~/.local/share/quickshell/quickshell.log
```

### Matugen colors not applying
```bash
bash ~/.local/bin/matugen-sync.sh ~/Pictures/wallpapers/current.jpg
cat /tmp/matugen/colors.json          # Verify colors generated
```

### Fonts not loading
```bash
fc-cache -fv                            # Rebuild font cache
fc-list | grep -i "jetbrains\|iosevka" # Verify installed
```

### MPD not responding
```bash
systemctl --user restart mpd.service
mpc status
```
